Build each menu item using ingredients from master inventory.

Import menu + inventory (Excel)

Upload a workbook with two sheets: Inventory and Menu. The Inventory sheet is processed first so all ingredients exist before menu recipes are linked. Missing categories are auto-created. Ingredients not found in the Inventory sheet are auto-created as placeholders.

Sheet 1 — Inventory

Columns: ingredient name, unit, unit cost, starting quantity

Each ingredient appears once here. All menu items that share the same ingredient reference the same stock entry — no duplicate quantity risk.

Sheet 2 — Menu

Columns: menu item name, category, price, ingredients, notes (optional)

Ingredient format: name|unit|recipeAmount. Separate multiple with ;.
Example: matcha powder|grams|3; milk|ml|240; cup|piece|1
